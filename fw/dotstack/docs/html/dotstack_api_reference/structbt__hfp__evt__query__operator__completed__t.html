Detailed Description

HFP_EVENT_QUERY_OPERATOR_COMPLETED event parameter.

HFP layer calls the session callback when it received operator's name from the AG during SLC setup or as a result of calling bt_hfp_hf_query_operator and passes a pointer to bt_hfp_evt_query_operator_completed_t which contains the name of the operator.

Detailed Description

Structure to store response to a "read by type" request.

An array of this structures is passed to bt_att_client_read_by_type(). It is application's responsibility to allocate memory for the array. When response is received the actual number of valid entries in the array is stored in bt_att_client_evt_read_by_type_response_t::count.

Detailed Description

Structure to store response to a "find by type value" request.

An array of this structures is passed to bt_att_client_find_by_type_value(). It is application's responsibility to allocate memory for the array. When response is received the actual number of valid entries in the array is stored in bt_att_client_evt_find_by_type_value_response_t::count.

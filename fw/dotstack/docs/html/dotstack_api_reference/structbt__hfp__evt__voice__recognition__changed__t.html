Detailed Description

HFP_EVENT_VOICE_RECOGNITION_CHANGED event parameter.

HFP layer calls the session callback when voice dialing in the AG has been turned on or off as a result of calling bt_hfp_hf_enable_voice_recognition. the callback is passed a pointer to bt_hfp_evt_voice_recognition_changed_t which contains the current status of voice dialing in the AG.

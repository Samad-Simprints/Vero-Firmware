Detailed Description

Parameter to the following events:

  • AVRCP_EVT_PLAYBACK_STATUS_CHANGED
  • AVRCP_EVT_TRACK_CHANGED
  • AVRCP_EVT_PLAYBACK_POS_CHANGED
  • AVRCP_EVT_BATT_STATUS_CHANGED
  • AVRCP_EVT_SYSTEM_STATUS_CHANGED
  • AVRCP_EVT_NOW_PLAYING_CONTENT_CHANGED
  • AVRCP_EVT_AVAILABLE_PLAYERS_CHANGED
  • AVRCP_EVT_ADDRESSED_PLAYER_CHANGED
  • AVRCP_EVT_UIDS_CHANGED
  • AVRCP_EVT_VOLUME_CHANGED

A pointer to this structure is passed to the AVRCP application callback as a valid member of the bt_avrcp_event_t union - bt_avrcp_event_t::notification - when a local device received one of the following notifications from the target:

  • Play status changed
  • Track changed changed
  • Playback position changed
  • Battery status changed
  • System status changed
  • Addressed player changed
  • UIDs changed
  • Volume changed
  • Player application setting changed The notification code defines which member of the bt_av_notification_t::params union is valid

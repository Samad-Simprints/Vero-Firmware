Detailed Description

Parameter to AVRCP_EVT_UIDS_CHANGED event.

A pointer to this structure is passed to the AVRCP application callback as a valid member of the bt_avrcp_event_t union - bt_avrcp_event_t::notification::params:uids - when a local device received a "UIDs changed" notification.

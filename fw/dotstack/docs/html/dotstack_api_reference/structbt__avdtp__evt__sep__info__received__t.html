Detailed Description

Parameter to AVDTP_EVT_SEP_INFO_RECEIVED event.

A pointer to this structure is passed to the AVDTP application callback as a valid member of the bt_avdtp_event_t union - bt_avdtp_event_t::sep_info_received - when AVDTP received positive result to a "discover" request. AVDTP_EVT_SEP_INFO_RECEIVED is generated for every SEP received from the remote device.

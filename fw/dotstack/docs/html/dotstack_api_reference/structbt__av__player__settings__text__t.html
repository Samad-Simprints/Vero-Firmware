Detailed Description

Parameter to AVRCP_EVT_PLAYER_SETTING_ATTRIBUTES_TEXT_RECEIVED event.

A pointer to this structure is passed to the AVRCP application callback as a valid member of the bt_avrcp_event_t union - bt_avrcp_event_t::player_settings_text - when a local device received a response to a "get player setting attributes displayable text" request.
